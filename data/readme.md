This dataset consists of images collected by the author and annotated for mangrove tree trunk detection and counting. There a total of 610 images split into train, validation, and test, with the ratio 7:2:1.

Due to dataset size and account-based control on Roboflow, the full dataset is not included in this repository.

### Dataset Structure (YOLO format)

images/
├── train/
├── val/
└── test/

labels/
├── train/
├── val/
└── test/

### Access
- The dataset was processed and versioned using Roboflow.
- Only a small sample is included under `data/` for demonstration.
- Dataset available upon request.
